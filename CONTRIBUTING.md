# Contributing

欢迎改进这个一级市场估值工具包。为保持估值结果可追溯，建议遵循以下原则：

1. 保持改动小而清晰，不要在一次提交里同时重写方法论、脚本和示例数据。
2. 新增估值方法时，同时补充配置字段、示例输入、输出说明和验证方式。
3. 不要把真实客户资料、未公开融资条款或敏感财务数据提交到仓库。
4. 所有报告结论应能追溯到配置文件、公开来源或明确的 analyst judgment。
5. 提交前至少使用 `assets/example-config-sujin.json` 跑通完整流水线。

本项目生成的报告可能被用于严肃决策场景，因此比“能跑”更重要的是：单位一致、来源清楚、假设透明、风险可见。
